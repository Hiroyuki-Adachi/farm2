== README

本システムは、営農組合向けの作業日報・経営管理プラットフォームです。
作業日報を軸に、人員・圃場・機械・農薬の利用実績を一元管理し、
戸別配分や日当、機械レンタル料、各種原価を自動集計できます。

日々の現場入力だけでなく、集計・照会・個人向け通知までを同じ基盤で扱えるため、
「記録するシステム」ではなく「経営判断に使えるデータを残すシステム」として運用できます。

主な機能:
- 作業日報登録（作業者・圃場・機械・農薬・備考）
- 戸別配分、日当、機械費、原価の集計
- 農薬在庫・棚卸、乾燥調製、育苗、契約、実績レポート管理
- QRログイン、個人向け画面、プッシュ通知などの現場連携

外部システム連携:
- Z-GIS
- ソリマチ会計
- Excel
- LINE

技術スタック:
- Ruby 3.3 系
- Rails 8.1 系
